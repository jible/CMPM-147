As they rise, they will turn green to show that they are at "grass level" then even higher up, they will turn brown, making them look like mountains. When right clicked, the tiles sink. If they sink below sand leve, they turn blue, indicating that they are now water. This is a fun way to make your own geolocial map! I'm pretty proud that I made it so the user can modify what ammount of tiles they control at once. I had to use the pythagorean theorem to measure how far each tile was, then I made a formula to make it so further tiles moved up at a slower rate ( it was just 1/distance from center).

Technical

I set the colors of all of the tiles to in a random range that would make them all turn out yellow. When the user clicks, it appends every modified tile to an array. Each point's index holds the height it should be at. non modified tiles are at the normal height. When tiles hit new height ranges, their random color range changes to be blue, green, or brown.
